
# Videojuego 2D con Canvas – Isaac Clicker

## Contexto

El presente proyecto fue desarrollado como parte del examen correspondiente a la Unidad II de la materia de Graficación. Su propósito es aplicar los conocimientos relacionados con la API Canvas de HTML5 para la construcción de un entorno gráfico interactivo en dos dimensiones, integrando elementos visuales, animación, interacción con el usuario y control de versiones mediante Git y GitHub.

---

## Objetivo

Desarrollar un videojuego 2D interactivo que implemente el uso de la API Canvas, cumpliendo con los requisitos establecidos, tales como la manipulación de múltiples objetos en pantalla, detección de colisiones, animaciones, interacción mediante eventos del mouse y el uso de recursos multimedia.

---

## Justificación

El desarrollo de este proyecto permite consolidar los conocimientos adquiridos en el área de graficación computacional, particularmente en el manejo de gráficos 2D en tiempo real. Asimismo, fortalece habilidades relacionadas con la lógica de programación, estructuración de proyectos web y el uso de herramientas de control de versiones, las cuales son fundamentales en el ámbito profesional del desarrollo de software.

---

## Operación del videojuego

El videojuego presenta un entorno interactivo en el que se visualizan 25 objetos en pantalla que representan enemigos. Cada uno de estos objetos se desplaza de manera independiente, utilizando distintos patrones de movimiento: horizontal, vertical, diagonal y circular.

El usuario puede interactuar con los objetos mediante clics del mouse. Al seleccionar un objeto, este desaparece temporalmente y reaparece en una nueva posición dentro del canvas, manteniendo constante la cantidad total de elementos.

El sistema incluye detección de colisiones entre los objetos, generando una animación visual que representa el impacto. Además, se incorpora un contador que registra la cantidad de objetos eliminados por el usuario.

El entorno del videojuego está complementado con una imagen de fondo, un cursor personalizado y música ambiental, lo cual contribuye a una mejor experiencia visual y de interacción.

## Conclusiones

El desarrollo de este proyecto permitió aplicar de manera integral los conceptos fundamentales de graficación en dos dimensiones mediante el uso de la API Canvas. Durante su implementación se presentaron desafíos relacionados con la detección precisa de colisiones, la correcta gestión del posicionamiento de los objetos y la integración de múltiples elementos interactivos en tiempo real.

Asimismo, se identificaron dificultades en el uso de herramientas de control de versiones, particularmente en la gestión de repositorios y resolución de errores en Git, lo cual evidenció la importancia de dominar estas tecnologías en el desarrollo de software.

En general, la actividad representó un refuerzo significativo de los conocimientos adquiridos previamente, permitiendo una mejor comprensión tanto de los aspectos técnicos como prácticos involucrados en el desarrollo de aplicaciones gráficas interactivas.
